# 转租


### 概念澄清

**客户：** 如果您与 ASMPT Virtual Assist 有直接的商业关系，则您是主要租户。这将使您能够将带有您的内容的白标应用程序转售给您的客户。这就是分租人概念发挥作用的地方。


**子租户：** 你的客户端是你的子租户，他们有权访问你添加并设置为“公共”的内容。尽管如果您的子租户添加自己的产品或产品线，他们将拥有对所有功能的完全访问权限，例如 **edit** :p encil2： 和 **delete** ：wastebasket：。

### 访问和可见性

**公开：** 如果您选择“公开”设置，则意味着有权访问该产品线的任何人（即使他们不属于您的组织）都能够看到您的内容，例如教程、文档或文章。


**专用：** 如果选择“专用”设置，则教程、文档或文章等内容将仅对组织内的人员可见，而对子租户不可见。这意味着只有那些被授予访问公司内部系统权限的人（例如员工或承包商）才能看到它。组织外部的任何人都无法访问本教程。
