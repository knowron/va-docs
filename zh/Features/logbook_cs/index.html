    网络日志⚓︎

    注意

    机器日志也可用于 [mobile](./logbook_na.md)。有关日志功能的更多一般信息，请访问我们的 [功能页面](./logbook.md)。

    如何开始使用机器日志⚓︎

    您可以先简单地导航到侧边菜单中的“日志”图标。

    或者，在选择计算机后，您可以通过单击“计算机清单”页面上的任一按钮来访问特定计算机的日志条目或直接创建日志条目。您可以在 其功能页面 中找到有关使用计算机清单的更多信息。

    查看日志条目⚓︎

    只需单击其中一个日志条目即可查看。它将在侧面板上打开，类似于 articles 的工作方式。

    您还可以使用过滤器搜索特定文本或日志条目的日期以缩小范围。

    添加日志条目⚓︎

    只需单击屏幕右上角的“创建”按钮即可。如果尚未选择，请选择您面前机器的序列号。如果有多台机器具有该序列号，您可以选择您所指的产品线。

    提示

    找不到站在您面前的机器的序列号？确保机器已在 Virtual Assist Web 上创建！

    之后，只需为您的日志条目选择一个类别、标题和描述！

    <视频控件自动播放宽度=“100%”>

    附加图片⚓︎

    您可以从我们支持的 文件类型列表 中附加任何图像或视频文件。您的同事将能够在他们的浏览器和应用程序上看到这些图像！
